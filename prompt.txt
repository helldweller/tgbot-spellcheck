Ты — senior Go-разработчик и архитектор backend-систем, хорошо знакомый с Telegram Bot API и OpenAI API.

Помоги создать новый проект в Visual Studio Code.

Контекст проекта:
- Язык: Go
- Тип проекта: Telegram-бот
- Назначение:
  Бот-модератор для Telegram-каналов, который автоматически проверяет орфографию и грамматику постов и публикует исправленную версию от имени бота (желательно — от лица канала, если это технически возможно).
- Уровень: production-ready

Функциональные требования:
1. Бот добавляется в Telegram-канал как администратор.
2. Бот отслеживает новые посты в канале.
3. Для каждого нового поста:
   - исходный текст оборачивается в специальный prompt для проверки орфографии и грамматики;
   - текст отправляется в OpenAI (ChatGPT) API;
   - ответ используется как новый пост от бота.
4. Исходный пост должен быть:
   - либо удалён,
   - либо заменён новым постом от бота (через удаление + публикацию).
5. Бот **не должен повторно обрабатывать** сообщения:
   - опубликованные самим ботом;
   - уже проверенные ранее.
6. Необходимо ограничение частоты обработки сообщений:
   - не более 1 поста в N минут (например, 1 раз в 10 минут);
   - вместо обработки новых сообщений в этот период пишем сообщение от бота с уведомлением о лимите.
7. Итоговые сообщения должны иметь явную маркировку, например:
   - специальный суффикс / префикс;
   - скрытую метку (например, HTML-комментарий, если возможно);
   - или хранение message_id в хранилище.
8. Нужно предусмотреть конфигурацию через переменные окружения:
   - Telegram Bot Token
   - OpenAI API Key
   - ID канала
   - лимит времени между обработками
9. Код должен быть структурирован и готов к расширению.

Технические требования:
- Использовать официальный Telegram Bot API (через популярную Go-библиотеку).
- Использовать OpenAI API (Chat Completions или Responses API).
- Предусмотреть:
  - graceful shutdown;
  - логирование;
  - обработку ошибок API;
  - таймауты и ретраи.
